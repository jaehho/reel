    Keys

    A mark is a piece of a clip worth keeping. Cut turns every mark into its own file, named after you.

    Marking

    hhighlight — mark the moment you just saw, in one press
    istart — begin a mark here instead, to pick both edges
    oend — close it at the playhead
    ename the last mark — the name lands in the cut filename
    uundo the last mark
    xdelete the selected mark — del and backspace do the same
    mshow every mark on this trip

    Watching

    spaceplay / pause
    jklshuttle back / stop / forward — press again to go faster
    ←→seek 5s, with shift 1s
    homeendjump to the start or end of this clip
    ctrlspaceplay just the marked segment, with shift on a loop
    ctrl←→jump between marks

    Trimming a mark

    With a segment up (ctrlshiftspace), these walk its edges and park you paused on that edge, so you see the exact frame it lands on.

    shift←→move the start earlier / later
    ctrl←→move the end earlier / later
    ctrlshiftspacepick the loop back up on the new range
    ioor jump that edge straight to the playhead
